# 💻 ReacTalk - Backend

Welcome to the **backend** of **ReacTalk**, a user registration and authentication server built with the **MERN Stack** backend technologies — **Node.js**, **Express**, and **MongoDB**. This backend handles user registration, password hashing, and exposes REST API endpoints for your frontend to consume.

---

## 🚀 Features

- ✅ User Registration API with validation
- ✅ Password hashing using bcryptjs
- ✅ MongoDB integration using Mongoose
- ✅ CORS enabled to allow frontend-backend communication
- ✅ Basic error handling and status codes
- 🔒 (Planned) User Authentication and JWT-based sessions
- 🔗 (Planned) Integration with frontend for full-stack flow

---

## 🛠️ Tech Stack

- **Node.js** & **Express.js**
- **MongoDB** & **Mongoose**
- **bcryptjs** for password security
- **cors** middleware for cross-origin requests
- **dotenv** for environment variables

---

## 📦 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ReacTalk.git
cd ReacTalk/backend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
nodemon server.js
```
